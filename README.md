# Number Guessing Game in C
A simple console-based Number Guessing Game developed using C programming language.

## About Project
The computer generates a random number, and the user tries to guess it.
The program provides hints whether the guessed number is higher or lower than the actual number.

## Features
- Random number generation
- User input handling
- Hint system (Higher/Lower)
- Attempt counter
- Simple console interface

## Technologies Used
- C Programming Language

## How to Run
1. Download the repository
2. Compile the C file:
gcc number_guessing_game.c -o game
3. Run the program:
./game
